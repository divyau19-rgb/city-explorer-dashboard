# Projects

A collection of full-stack web applications built for learning and portfolio purposes.

## Apps

| Folder | Description | Stack |
|---|---|---|
| [`city-explorer/`](./city-explorer) | Live city dashboard — weather, air quality, events, favourites | Next.js 16, SQLite, Open-Meteo |

## Running an app

Each app is self-contained. Navigate into its folder and follow its own README:

```bash
cd city-explorer
npm install
npm run dev
```
